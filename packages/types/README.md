# @enterprise-ai/types

## Purpose

The `@enterprise-ai/types` package serves as the single source of truth for TypeScript interfaces, data transfer objects (DTOs), domain models, and API response structures across the monorepo.

---

## 🎯 Planned Responsibilities

- **Domain Models**: User, Tenant, Role, Agent, Document, VectorIndex, and AuditLog interfaces.
- **API Payload DTOs**: Request body, query parameter, and standardized API envelope response types.
- **AI Engine Interfaces**: Agent state definitions, tool schema interfaces, and LLM message payload contracts.
