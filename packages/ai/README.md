# @enterprise-ai/ai

## Purpose

The `@enterprise-ai/ai` package is the core orchestration module defining agent state graphs, tool registries, vector database connectors, and LLM provider interfaces.

---

## 🎯 Planned Responsibilities

- **LangGraph Agent Engine**: Stateful workflow definitions, conditional edge execution, memory checkpointing, and human-in-the-loop validation nodes.
- **RAG & Vector Retrieval**: Embedding pipeline connectors, Qdrant client wrappers, and hybrid search (sparse + dense) retrieval modules.
- **LLM Abstraction Layer**: Unified model provider adapters for local privacy-first Ollama models and cloud provider APIs.
- **Prompt & Tool Registry**: Versioned prompt templates and tool definitions with JSON schema parameter validation.

---

## 🛑 Current Status

Workspace placeholder. LangGraph, LangChain, and Qdrant integration modules will be added in subsequent implementation phases. No application code or dependencies exist in this package yet.
