# Deployment Infrastructure & Orchestration

## Purpose

The `infrastructure/` directory contains configuration blueprints for local containerized development and cloud production deployment.

---

## 🎯 Planned Responsibilities

- **Local Infrastructure (`infrastructure/docker/`)**: Docker Compose manifests provisioning PostgreSQL 16, Qdrant Vector Store, and local Ollama LLM runtime.
- **Kubernetes Production (`infrastructure/k8s/`)**: Helm charts, ingress controllers, horizontal pod autoscalers (HPA), and secrets management.
- **Environment Management**: Infrastructure as Code (IaC) templates for cloud provisioning.
